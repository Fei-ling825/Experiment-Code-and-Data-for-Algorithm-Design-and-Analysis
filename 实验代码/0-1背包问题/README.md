# 0-1 背包问题 — C 语言实现（无规模限制）

## 编译运行

```bat
cd D:\算法\0-1背包问题\c
build.bat
```

## 实验配置（严格按题目要求）

- **物品数量**: 1000, 2000, 3000, 4000, 5000, …, 320000（共 15 档）
- **背包容量**: 10000, 100000, 1000000
- **重量**: 1~100 随机整数
- **价值**: 100~1000 随机数，保留两位小数
- **四种算法均无人工规模限制**

## 两套实验

| 实验 | 输出文件 | 说明 |
|------|----------|------|
| **小规模四算法对比** | `output/small_scale_results.csv` | n=10~28, W=500，四种算法全部运行 |
| **大规模主实验** | `output/experiment_results.csv` | 题目要求规模；n>300 跳过蛮力，n>1500 跳过回溯 |

## 生成图表

```bat
python plot_results.py
```

会同时生成：
- `small_scale_comparison.png` — 四算法完整对比（时间 + 空间）
- `time_comparison_W*.png` / `space_comparison_all.png` — 大规模实验
- `表1_0-1背包问题1000个物品统计信息.csv` 或备用 `table1_items_1000.csv` — 表 1 数据

## 说明

- 小规模实验用于观察蛮力法/回溯法的指数增长特征
- 大规模实验中，蛮力法 n>300、回溯法 n>1500 自动跳过并记录原因
- 动态规划在 n×W 极大时可能内存不足（仅报告分配失败）
- 每完成一组即写入 CSV（`fflush`），中断后仍保留已有结果
